# Use Cases

## 1. Code Explanation

A developer can ask:

"Explain how the authentication system works."

The agent reads project files and generates a structured explanation.

---

## 2. Code Refactoring

A developer can ask:

"Refactor this module to use dependency injection."

The agent reads files and suggests edits.

---

## 3. Debugging

A developer can ask:

"Why does this API return a 500 error?"

The agent analyzes logs and code.

---

## 4. Repository Navigation

A developer can ask:

"Find where user authentication is implemented."

The agent searches the repository.

---

## 5. Developer Automation

Examples:

* run tests
* build project
* check logs
* run scripts
